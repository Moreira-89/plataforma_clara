# plataforma_clara
A Plataforma Clara é uma solução para aumentar a transparência nos FIDCs, permitindo que investidores e gestoras vejam informações claras sobre aportes e alocações e acessem um Ranking de Reputação das empresas financiadas, criado com Machine Learning. Desenvolvida em contexto acadêmico, com foco em robustez e baixo custo para MVP.
